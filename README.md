# Final-team-project-flask

AI 기반 음식 이미지 분석 서버 - Flask AI 서버

## 개발환경

Java 17, Spring Boot 3.5.4, Flutter SDK 3.9.2, MariaDB, MongoDB, QueryDSL 5.0.0, JWT, Flask 2.3.0, PyTorch 2.0.0, Google Maps API, YouTube API v3, OAuth2 (Google, Naver)

## 개요

Flask 기반 AI 서버로, EfficientNet 모델을 사용하여 음식 이미지를 분석하고 Top-3 예측 결과를 제공합니다.

## 주요 기능

- 음식 이미지 분석 (EfficientNet 모델)
- Top-3 예측 결과 반환
- JSON 형식 API 응답

## 기술 스택

- Flask 2.3.0
- PyTorch 2.0.0
- Torchvision 0.15.0
- Transformers 4.30.0
- Pillow 10.0.0

## 설치 및 실행

```bash
pip install -r requirements.txt
python app.py
```
